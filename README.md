# Дневник

Локальный ежедневник в стиле Stanza Terminal. Проект начинается как простое приложение для ежедневных записей, но развивается в полноценный личный инструмент: календарь, история, статистика, автоматизация, приватность и PWA.

## Текущий статус

Версия: `0.1.0`

Сейчас работает:

- запись на выбранную дату;
- календарь месяца;
- история записей;
- поиск и фильтры;
- статистика;
- настройки;
- импорт и экспорт JSON;
- статус автосохранения;
- шаблон дня;
- недельный обзор;
- локальное хранение в браузере;
- PWA-заготовка.

AI-функции намеренно отложены. Сначала проект должен получить устойчивую архитектуру, приватность и надёжное хранение.

## Как открыть

Можно открыть `index.html` напрямую в браузере.

Для PWA-режима лучше запускать через локальный сервер:

```bash
npm run dev
```

После этого открыть:

```text
http://localhost:4173
```
