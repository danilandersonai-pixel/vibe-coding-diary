# Changelog

Формат близок к Keep a Changelog. Версии пока проектные, без Git-тегов.

## 0.1.0 - 2026-05-16

Первая рабочая версия.

Добавлено:

- статус автосохранения с состояниями `сохраняю`, `сохранено`, `ошибка`;
- шаблон дня для пустых полей записи;
- недельный обзор по локальным данным без AI;
- мобильный интерфейс по референсам Stanza Terminal;
- экран входа;
- экран записи;
- календарь месяца;
- история записей;
- статистика;
- настройки;
- редактирование записей;
- импорт и экспорт JSON;
- локальное хранение записей;
- PWA-заготовка: `manifest.webmanifest`, `sw.js`, `icon.svg`;
- модальное подтверждение вместо системного `confirm`;
- файл идей `docs/IDEAS.md`;
- план развития `ROADMAP.md`;
- описание архитектуры `docs/ARCHITECTURE.md`;
- модель данных `docs/DATA_MODEL.md`;
- архитектурные решения `docs/DECISIONS.md`;
- план внедрения `docs/IMPLEMENTATION_PLAN.md`;
- модель приватности `docs/PRIVACY.md`;
- `package.json` со скриптами запуска и проверки;
- README проекта.

Отложено:

- AI-агенты;
- подключение внешних ключей;
- локальное шифрование;
- полноценное IndexedDB-хранилище.
